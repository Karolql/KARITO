Aplicación del Desarrollo Rápido de Aplicaciones (RAD)

El modelo RAD se enfoca en el desarrollo rápido y la entrega temprana de prototipos funcionales mediante herramientas visuales. Este enfoque se complementa con la programación orientada a eventos, facilitando la creación de interfaces interactivas.

Fases de RAD en este submódulo

  1. Modelado del negocio: Diseño de formularios y pantallas para entender el flujo del usuario.
  2. Modelado de datos: Crear estructuras simples para almacenar información (bases de datos o listas).
  3. Generación de interfaces: Uso de herramientas visuales como Visual Studio, App Inventor o Blockly.
  4. Codificación orientada a eventos: Programar respuestas a acciones como clics, teclas, o desplazamiento.

Ejemplo de aplicación

Proyecto: Aplicación de registro de asistencia escolar.

  • Diseño visual con botones, cajas de texto y etiquetas.
  • Eventos como onClick, onChange para registrar asistencia.
  • Prototipo funcional en una semana usando Visual Studio o App Inventor.
